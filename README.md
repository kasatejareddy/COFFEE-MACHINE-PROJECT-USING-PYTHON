# ☕ Coffee Machine Project

A Python-based Coffee Machine simulator that allows users to order coffee, insert coins, check available resources, and generate reports.

## Features

* ☕ Order Espresso, Latte, or Cappuccino
* 💰 Coin-based payment system
* 💵 Automatic change calculation
* 📊 Resource and profit reporting
* ✅ Resource availability checks
* 🚫 Invalid input handling
* 🔌 Turn the machine off with the `off` command

## Menu

| Drink      | Cost  |
| ---------- | ----- |
| Espresso   | $1.50 |
| Latte      | $2.50 |
| Cappuccino | $3.00 |

## Available Commands

* `espresso` - Order an espresso
* `latte` - Order a latte
* `cappuccino` - Order a cappuccino
* `report` - Display current resources and profit
* `off` - Turn off the coffee machine

## Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements
* User Input Handling

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd coffee-machine-project
   ```

3. Run the program:

   ```bash
   python main.py
   ```
